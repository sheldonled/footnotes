Foot Notes 0.1

Foot Notes is a piece of Javascript (jQuery) HTML and CSS code, that give links a good feature that, when you click on it shows up a box with featured information about the link. It is a plus for User Experience and makes your site more beauty.

The development of this tool ocurred this way: Sheldon Led saw this feature in Zeno Rocha's website and liked it. So he got the code and adapted for his own website. After this he put the code on github.

All you have to do is set up the Javascript and CSS code, so when you want a foot note link, just put <a href='#fn:1' rel='ftnote'> tag on your link. The 1 is a counting of references, starting with 1 to N references. So down in your HTML document, put a div with "footnotes" class, an then an <ol> list. You can see the HTML code of this page, to understand.
