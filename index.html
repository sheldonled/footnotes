Foot Notes 1.0

About

Foot Notes is a piece of Javascript (jQuery) CSS code, that give links a good feature that, when you click on it shows up a box with featured information about the link. It is a plus for User Experience and makes your site more beauty.

How to use it?

  • Make sure your website have jQuery.
  • Download the code
  • Copy and link footnotes.js
  • Copy and link footnotes.css
  • Now, to make a Foot Note, just:
    • put this code in your HTML
      •                                     <div id="mask"></div>
    • Write a link like this: 
                                          <a href='#fn:1' rel='ftnote'> Text Inside </a>
    • And now you have to write a code like this: 
                                          <div class='footnotes'>
                                        <hr />
                                        <ol>
                                            <li id='fn:1'>
                                                <p><strong>Foot Notes</strong>
                                                    When you click on a link the information shows up in a 
                                                    box like this :D
                                                </p>
                                            </li>
                                        </ol>
                                    </div>
  • The 1 in #fn:1, is a counting of references, starting with 1 to N references. So down in your HTML document, put a div with "footnotes" class, an then an <ol> list. You can see the HTML source code of this page, to get a real example.
